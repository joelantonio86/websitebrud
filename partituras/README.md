# Pasta de Partituras

Coloque aqui todos os arquivos PDF das partituras das músicas.

## Estrutura Sugerida

```
partituras/
├── apoteose-racional/
│   ├── fuzileiro.pdf
│   ├── atabaque.pdf
│   ├── caixa.pdf
│   └── surdo.pdf
├── marcha-racional/
│   ├── fuzileiro.pdf
│   ├── atabaque.pdf
│   ├── caixa.pdf
│   └── surdo.pdf
└── ...
```

## Instrumentos

As partituras são organizadas por instrumento:

- **Fuzileiro**: Partitura para fuzileiro
- **Atabaque**: Partitura para atabaque
- **Caixa**: Partitura para caixa
- **Surdo**: Partitura para surdo

## Formato

- Use formato PDF para as partituras
- Mantenha os nomes dos arquivos consistentes
- Atualize os links no HTML quando adicionar novas partituras
