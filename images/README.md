# Pasta de Imagens

Coloque aqui todas as imagens do site da banda.

## Estrutura Sugerida

```
images/
├── banda/
│   ├── foto-banda-1.jpg
│   ├── foto-banda-2.jpg
│   └── ...
├── shows/
│   ├── show-1.jpg
│   ├── show-2.jpg
│   └── ...
├── musica/
│   ├── capa-musica-1.jpg
│   ├── capa-musica-2.jpg
│   └── ...
└── galeria/
    ├── galeria-1.jpg
    ├── galeria-2.jpg
    └── ...
```

## Formatos Recomendados

- **Fotos**: JPG ou PNG
- **Logos**: PNG com fundo transparente
- **Capa de álbuns**: JPG ou PNG (formato quadrado 1:1)
- **Fotos de shows**: JPG

## Tamanhos Sugeridos

- **Fotos da banda**: 1200x1200px ou maior
- **Capa de músicas**: 1000x1000px (quadrado)
- **Fotos de galeria**: 800x800px ou maior
- **Banner hero**: 1920x1080px

## Otimização

Antes de adicionar as imagens, considere otimizá-las para web:
- Use ferramentas como TinyPNG ou ImageOptim
- Mantenha o tamanho do arquivo pequeno para melhor performance
- Use formato WebP quando possível para melhor compressão
